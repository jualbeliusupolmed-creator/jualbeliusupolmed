# Panduan Cepat — Jual Beli USU Polmed

**Website:** jualbeliusupolmed.web.id  
**WhatsApp Bot:** hubungi nomor yang tertera di website

---

## Cara Pasang Iklan

### Via WhatsApp (paling mudah)
1. Kirim **foto barang** ke nomor WA bot
2. Sertakan **deskripsi, harga, dan kondisi** di caption foto
3. AI akan baca iklanmu otomatis → konfirmasi detail
4. **Scan QRIS** untuk bayar biaya pasang
5. Kirim **screenshot struk** → iklan langsung tayang ✅

### Via Website
1. Buka **jualbeliusupolmed.web.id/jual**
2. Login dengan nomor WA (OTP dikirim ke WA)
3. Isi form iklan + upload foto
4. Bayar QRIS + upload screenshot struk
5. Iklan tayang ✅

> Iklan aktif selama **14 hari**. Kode iklan dikirim ke WA setelah tayang.

## Perintah WA Penting

| Perintah | Fungsi |
|---|---|
| `MENU` | Daftar semua perintah |
| `SAYA` | Lihat profil, iklan & statistikmu |
| `NAMA [nama baru]` | Ganti nama profilmu |
| `IKLANKU` | Daftar semua iklanmu |
| `CEK` | Cek semua iklan aktifmu |
| `CEK 1001` | Cek status iklan tertentu |
| `BUMP 1001` | Naikkan iklan ke atas (bayar) |
| `PERPANJANG 1001` | Perpanjang masa iklan (bayar) |
| `UPGRADE FEATURED 1001 3` | Iklan unggulan 3 hari |
| `UPGRADE AUTOBUMP 1001` | Auto bump 7 hari |
| `HAPUS LAKU 1001` | Tandai barang sudah terjual |
| `HAPUS GALAKU 1001` | Minta hapus iklan |
| `TAWAR 1001 150000` | Tawar harga iklan orang lain |
| `TAWARAN` | Lihat tawaran masuk ke iklanmu |
| `TAWAR BIAYA 1001 5000` | Negosiasi biaya pasang iklan ke admin |
| `DICARI laptop 3 juta` | Post barang yang kamu cari |
| `LANGGANAN Elektronik` | Notif WA tiap ada iklan baru |
| `LAPOR 1001 alasan` | Laporkan iklan mencurigakan |

> Ganti `1001` dengan **kode iklan** (angka yang dikirim bot saat iklan dibuat)

## Tips

- Foto **jelas & pencahayaan bagus** = lebih banyak dilihat
- Tulis harga, kondisi, dan lokasi di caption foto
- Set nama profilmu dengan `NAMA [nama]` agar pembeli percaya
- Ketik `BUMP 1001` kalau iklan mau naik ke atas lagi
- Balas tawaran pembeli lewat link WA yang dikirim bot
- Cek dashboard di website untuk statistik lengkap: **jualbeliusupolmed.web.id/dashboard**
